# 📲 SMS შეტყობინებები — სრული დაყენების ინსტრუქცია

ფუნქციონალი:
1. ✅ ადმინების SMS ახალი რეგისტრაციისას (ნებისმიერი რაოდენობა)
2. 🔔 შეხსენება მონაწილეებს ღონისძიებამდე — **დროის რეგულირება Sheet-დან**
3. 🚌 შეხსენება ავტობუსის გასვლამდე — **დროის რეგულირება Sheet-დან**

---

## 🎛️ Sheet-ის Config-ში დამატებული ცვლადები

დაამატე ეს ცვლადები შენს `Config` ფურცელში (A სვეტში). თუ ცარიელი დატოვებ — გამოიყენება default მნიშვნელობა.

| A სვეტი | B სვეტი (მაგალითი) | აღწერა | Default |
|---------|--------------------|--------|---------|
| `admin_phones` | `599123456, 555987654` | ადმინების ტელეფონები (მძიმეებით) | (ცარიელი = არ გაუგზავნის) |
| `reminder_days_before` | `1` | რამდენი დღით ადრე გაიგზავნოს შეხსენება | `1` (ხვალ) |
| `reminder_hours_before` | `3` | რამდენი საათით ადრე გაიგზავნოს ავტობუსის შეხსენება | `3` |

### `reminder_days_before` ვარიანტები

- `1` → "ხვალ Farmasi-ის მასტერკლასია" (ნაგულისხმევი)
- `2` → "ზეგ Farmasi-ის მასტერკლასია"
- `3` → "მაზეგ Farmasi-ის მასტერკლასია"
- `7` → "7 დღეში Farmasi-ის მასტერკლასია"

### `reminder_hours_before` ვარიანტები

- `3` → "🚌 ავტობუსი 3 საათში!" (ნაგულისხმევი)
- `2` → "🚌 ავტობუსი 2 საათში!"
- `1.5` → "🚌 ავტობუსი 1.5 საათში!"
- `0.5` → "🚌 ავტობუსი 0.5 საათში!" (30 წუთი)

## 📨 1. ადმინების SMS — ავტომატურად

ყოველი ახალი რეგისტრაციისას ადმინებს მოდის:

```
🆕 ახალი რეგისტრაცია #25
🌟 თენგიზ ბერიძე
📞 599123456
🪑 ადგილი #15 · Early Bird
🎫 FM-2026-00025
```

ნომრები ნებისმიერ ფორმატში — `599123456`, `+995 599 12 34 56`, `0599 12 34 56` — ავტომატურად ნორმალიზდება.

---

## 🔔 2. შეხსენება ღონისძიებამდე

**ფუნქცია:** `sendReminderDayBefore()`
**დრო:** Sheet-ში `reminder_days_before` (default: 1 დღე)

### Trigger-ის დაყენება

1. Apps Script Editor → **⏰ Triggers** (მარცხნივ)
2. **+ Add Trigger** (ქვემო-მარჯვნივ)
3. შეავსე:
   - **Function:** `sendReminderDayBefore`
   - **Event source:** Time-driven
   - **Type:** Day timer
   - **Time of day:** 9am-10am (ერთხელ დღეში)
4. **Save**

ფუნქცია ყოველდღე ერთხელ მოწმდება. თუ:
- **დღევანდელი + `reminder_days_before` = ღონისძიების თარიღი**

მაშინ ყველა მონაწილეს SMS-ი გაუგზავნის.

---

## 🚌 3. შეხსენება ავტობუსის გასვლამდე

**ფუნქცია:** `sendReminderBusDeparture()`
**დრო:** Sheet-ში `reminder_hours_before` (default: 3 საათი)

### Trigger-ის დაყენება

1. Apps Script Editor → **⏰ Triggers**
2. **+ Add Trigger**
3. შეავსე:
   - **Function:** `sendReminderBusDeparture`
   - **Event source:** Time-driven
   - **Type:** Hour timer
   - **Hour interval:** Every hour
4. **Save**

ფუნქცია ყოველ საათში მოწმდება. თუ:
- ღონისძიების დღეა **და**
- მიმდინარე საათი = `transport_time` − `reminder_hours_before` (±30 წუთი)

მაშინ ყველა მონაწილეს SMS-ი გაუგზავნის.

**მაგალითი:**
- `transport_time = 12:00`
- `reminder_hours_before = 3`
- SMS გაიგზავნება **9:00-9:30 ფანჯარაში**
